# React Dashboard UI

A modern dashboard application built using React, Vite, and Tailwind CSS. This project demonstrates component-based architecture, API integration, and clean UI design.

---

## 🚀 Features

- Sidebar navigation with active state
- Navbar with search bar and profile section
- Reusable Stats Card components
- Dynamic table with API data
- Loading and error handling
- Clean and responsive layout

---

## 🛠️ Tech Stack

- React (Functional Components & Hooks)
- Vite
- Tailwind CSS
- JavaScript (ES6+)

---

npm install
